# FLIPMAX

FLIPMAX is a static browser UI for a demo arcade. It includes no login, payments, deposits, withdrawals, or real-money gambling flow.

Games included:

- Mines: 5x5 grid, 3 hidden mines, cash out after safe picks.
- Roulette: red and black pay 2x, green pays 14x.
- Rocket Multiplier: live multiplier climbs until a hidden crash point.
- Plinko: chip drop lands in a payout multiplier bucket.
- Towers: climb safe rows and cash out before a miss.

Open `index.html` in a browser to run it.
